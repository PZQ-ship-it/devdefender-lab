# DevDefender Lab

Phase 1 code defense smoke deck

---

## Code Graph

- Parses Python files with tree-sitter
- Persists functions, imports, and call edges
- Keeps source pointers instead of full repository text

---

## Defense Loop

- Typed reviewer feedback wakes the graph flow
- OpenAI drafts a grounded defense
- Structured issue JSON captures actionable change requests
